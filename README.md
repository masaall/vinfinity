A minimal 64-bit (x86-64) Bare-Metal OS inspired by xv6, build from scratch just for fun.

Feature:
  1. Custom bootloader
  2. 64-bit kernel
  3. VGA text mode
  4. Physical memory allocator
  5. 5-level paging (PML5 -> PML4 -> PDP -> PD -> PT)
